The Greatest Debugging Suite

Dex+++ is your go-to explorer, specifically designed to help you debug or identify vulnerabilities within Roblox games. Originally created by Moon back around 2017 to 2018, Dex has seen many iterations over the years. Dex+++ stands out as one of the latest versions, a feature rich, modified edition of Dex++ developed by Chillz.

What's inside? ✨

Everything you need to poke around a game, all in one cozy suite.

📂
Advanced Object Explorer

A speedy virtual tree that renders massive instance hierarchies without lag. Real time instance tracking plus deep nested search and dynamic filtering.

📊
Dynamic Property Inspector

Inspect and tweak reflection data on the fly. Changed signal listeners track state mutations the instant they happen.

📐
3D Model Viewer

A cozy isolated viewport to pull out any model and study its geometry and spatial data away from the busy workspace.

🔒
Inline Script Obfuscator

Protect your runtime strings and generated logic right inside the environment with localized AST transformations.

💻
Developer Console

A clean, low overhead log engine that catches output, errors and warnings with precise microsecond timestamps.

🕵️
Embedded Network Spy

Intercepts and decodes RemoteEvent and RemoteFunction calls, with tidy data serialization for quick payload analysis.

💬
Real Time Chat Logger

Quietly captures incoming and outgoing chat, whispers, system logs and channel messages with source indexing.

💾
High Efficiency Save Instance

A fully overhauled serialization engine that skips the slow default dumping. Structural compression means smaller files and way less disk writing.

📝
Integrated Notepad

A little scratchpad for hot swapping code snippets, jotting down memory addresses, or staging scripts mid debugging session.

🔮
Infinite Yield Commands

The beloved Infinite Yield command suite, baked right into Dex+++ so all your favorite commands live one window away.

🎶
Audio Logger

Peek at the ID of any audio playing in the game, so you never have to wonder what track is looping in the background.

Getting started 🚀

1

Copy the loadstring with the big rainbow button up top. It lands right in your clipboard.

2

Paste it into your executor of choice while in a Roblox game.

3

Run it and the Dex+++ window pops up, with the whole game tree yours to explore. Happy debugging! 💙
